# Инструкция для AI-ассистентов (Claude Code / Cursor / и т.д.)

Этот файл коммитится в репо. Когда пользователь открывает проект — AI читает этот файл и подстраивается под правила команды.

## О проекте

- 2D top-down MMORPG.
- Стек: **Godot 4** (клиент) + **Nakama** (сервер).
- Этап: миграция с Phaser+Colyseus на Godot+Nakama. Фаза 1 — Godot-скелет.
- Репозиторий: https://github.com/DmitryLL/GG
- **Не упоминай в коде/коммитах/документах название референс-игры** (ту, на которую проект похож). Это жёсткое правило команды.

## Общие правила для обоих

1. **Язык:** чат и коммиты — по-русски. Комментарии в коде — по-русски.
2. **Коммиты:** короткие, осмысленные, через PR в `main`. Прямой push в `main` запрещён настройкой ветки.
3. **Never force push / reset --hard** без явной команды пользователя.
4. **Никогда не коммить секреты:** `export_credentials.cfg`, `.env`, `*.sqlite`, API-ключи.
5. **Бинари >1 МБ** прогоняй через сжатие (pngquant для PNG).
6. **Не создавай лишние `.md`-файлы** если пользователь не просил. Этот файл, `AGENTS.md` и README уже покрывают большинство случаев.
7. **Локальный оверрайд:** если у тебя/пользователя есть личные заметки, клади их в `CLAUDE.local.md` (в gitignore) — не ломай коммитящийся `CLAUDE.md`.
