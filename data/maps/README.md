# Карты мира

Tiled-карты в JSON формате (`.tmj`). Сервер вшивает в бандл, клиент
читает из `assets/`.

## Файлы

| Файл | Что |
|---|---|
| `tiles.tsj` | Описание тайлсета — какие тайлы есть и какие проходимы. |
| `tiles.png` | Сама графика (копия из `godot/assets/sprites/tiles.png`). |
| `world.tmj` | Текущая карта мира. Главная — её и редактируешь. |
| `generate_initial.py` | Скрипт, который создаёт `world.tmj` из старого процгена. Запускать только если хочешь сбросить карту до стартовой. |

## Как редактировать в Tiled

1. Скачай [Tiled](https://www.mapeditor.org/) (бесплатно, Mac/Win/Linux).
2. **File → Open** → выбрать `world.tmj`.
3. Слева видно три слоя:
   - **Tiles** — сам мир (тайлы трава/вода/деревья/камень/песок/дорожка).
   - **Mobs** — точки спавна мобов (квадратные «штампы», `type = slime` или `type = goblin`).
   - **NPCs** — точка спавна игрока (`name = player_spawn`) и торговец.
4. Чтобы рисовать тайлы:
   - Выбрать слой **Tiles** в правой панели.
   - В нижней панели выбрать тайл из тайлсета (трава/вода/etc).
   - Левый клик по карте — поставить, правый — стереть.
5. Чтобы добавить моба:
   - Выбрать слой **Mobs**.
   - Тип объекта **Insert Rectangle** (или дублировать существующий).
   - В правой панели поставить **Type** = `slime` или `goblin`.
   - Перетащить на нужное место.
6. **File → Save**.

## Как залить в игру

```
git add data/maps/world.tmj
git commit -m "новая локация: пустынный оазис"
git push
```

Через ~3 минуты CI пересобирает Nakama-модуль и Godot-клиент.
Сервер перезапускается, на VPS появляется новая карта.

## Что нельзя ломать

- Размер карты **60×45** (в `balance.json` и в `tiles.tsj` — должно совпадать).
- Имена слоёв `Tiles`, `Mobs`, `NPCs` (сервер ищет по ним).
- `name=player_spawn` обязателен — иначе игрок появится в (960, 720) по умолчанию.
- Тайл-ID — добавить новый тип тайла можно только обновив `tiles.tsj` + код walkability.

## Если хочешь сбросить карту

```
python3 data/maps/generate_initial.py
```

Перезапишет `world.tmj` стартовой процедурной картой.
